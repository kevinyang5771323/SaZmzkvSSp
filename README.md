# 个性化餐饮场所推荐系统 python462

## 项目概述

本项目是一款基于Python开发的个性化餐饮场所推荐系统，通过结合Vue前端框架和Mysql数据库，采用Django框架进行后端开发，致力于为用户提供优质的餐饮推荐服务。

## 技术栈

- **后端：**
  - Python
  - Django
- **前端：**
  - Vue
- **数据库：**
  - Mysql 8

## 系统角色

- 用户：浏览和获取个性化餐饮场所推荐
- 管理员：负责系统维护和数据处理

## 运行环境

- **操作系统：** Windows 10 / MacOs
- **Python版本：** 3.7.7
- **数据库：** Mysql 8

## 部署步骤

1. 安装Mysql 8数据库
2. 准备Python 3.7.7环境
3. 使用pycharm导入项目
4. 安装依赖包
5. 初始化数据库和表结构
6. 部署前端Vue项目

## 目录结构

```
project/
│

│   ├── apps/               # Django应用目录

│   ├── views/              # 页面目录
│   └── ...
│
├── database/               # 数据库相关文件
└── ...
```

## 核心亮点

- **个性化推荐算法：** 根据用户行为和偏好提供定制化餐饮推荐
- **MVC架构模式：** 系统结构清晰，易于维护和扩展
